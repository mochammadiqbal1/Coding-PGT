#Date

Menampilkan atau menetapkan tanggal sistem. Jika digunakan tanpa parameter, tanggal menampilkan pengaturan tanggal sistem saat ini dan meminta Anda untuk memasukkan tanggal baru.

#Color

Mengubah warna latar depan dan latar belakang di jendela Prompt Perintah untuk sesi saat ini. Jika digunakan tanpa parameter, warna akan mengembalikan warna latar depan dan latar belakang jendela Command Prompt default.

#Cls

Membersihkan jendela Prompt Perintah.

#Exit

Keluar dari jendela Prompt.

#Cd

Menampilkan nama direktori saat ini atau mengubah direktori saat ini.

#Echo

Menampilkan pesan atau mengaktifkan atau menonaktifkan fitur echo perintah. Jika digunakan tanpa parameter, echo menampilkan pengaturan echo saat ini.

#Erase

Menghapus satu atau lebih file. Jika Anda menggunakan erase untuk menghapus file dari disk, Anda tidak dapat mengambilnya.

#Edit

Mengubah file teks.


#pause

Untuk memberhentikan code progam sementara.

#start

Untuk memulai code progam.

#verify

Memberi tahu alat Prompt Perintah (cmd.exe) apakah akan memverifikasi file Anda ditulis dengan benar ke disk.

#Dir

Menampilkan daftar file dan subdirektori direktori. Jika digunakan tanpa parameter, perintah ini menampilkan label volume dan nomor seri disk, diikuti dengan daftar direktori dan file pada disk (termasuk nama dan tanggal serta waktu masing-masing terakhir diubah). Untuk file, perintah ini menampilkan ekstensi nama dan ukurannya dalam byte. Perintah ini juga menampilkan jumlah total file dan direktori yang terdaftar, ukuran kumulatifnya, dan ruang kosong (dalam byte) yang tersisa pada disk.


#Tree

Menampilkan struktur direktori jalur atau disk dalam drive secara grafis. Struktur yang ditampilkan oleh perintah ini bergantung pada parameter yang Anda tentukan pada prompt perintah. Jika Anda tidak menentukan drive atau jalur, perintah ini menampilkan struktur pohon yang dimulai dengan direktori saat ini dari drive saat ini.


#Ping

Memverifikasi konektivitas tingkat IP ke komputer TCP/IP lain dengan mengirimkan pesan permintaan gema Internet Control Message Protocol (ICMP). Tanda terima pesan Balasan gema yang sesuai ditampilkan, bersama dengan waktu pulang pergi. ping adalah perintah TCP/IP utama yang digunakan untuk memecahkan masalah konektivitas, keterjangkauan, dan resolusi nama. Digunakan tanpa parameter, perintah ini menampilkan konten Bantuan.


#Ipconfig

Menampilkan semua nilai konfigurasi jaringan TCP/IP saat ini dan menyegarkan pengaturan Protokol Konfigurasi Host Dinamis (DHCP) dan Sistem Nama Domain (DNS). Digunakan tanpa parameter, ipconfig menampilkan Internet Protocol versi 4 (IPv4) dan alamat IPv6, subnet mask, dan gateway default untuk semua adapter.


#Mkdir

Membuat direktori atau subdirektori. Ekstensi perintah, yang diaktifkan secara default, memungkinkan Anda menggunakan satu perintah mkdir untuk membuat direktori perantara di jalur tertentu.


#Move

Memindahkan satu atau lebih file dari satu direktori ke direktori lain.


#Rename

Mengganti nama file atau direktori.


#Shutdown

Memungkinkan Anda untuk mematikan atau memulai ulang komputer lokal atau jarak jauh, satu per satu.
